# 🤖 K9 Controller // DR301 REACTIVATION PROTOCOL

> _"Affirmative, Master."_  

A modern rewrite of the original **2012 DR301 app by Zeon** for their  
**smartphone-controlled K9 unit**.

After the official app was discontinued, compatibility with modern devices was lost
rendering many K9 units non-functional.

This project restores control.  
This project brings K9 back online.

---

## 🚀 ACTIVATION SEQUENCE

Follow these steps to reinitialize your K9 unit:

### 1 ▸ POWER CORE ONLINE
Hold the power button until K9 activates.

### 2 ▸ PAIRING MODE
Wait for the eye to begin **flashing red**.

### 3 ▸ BLUETOOTH LINK
Connect to K9 via your device's Bluetooth settings.

### 4 ▸ INITIALIZE INTERFACE
Launch the application.

### 5 ▸ AUDIO CALIBRATION
Set volume to **100%** for optimal sound transmission.
